# MentorMod Privacy Policy

Last updated: January 2026

This Privacy Policy explains how MentorMod processes data.

## 1. Data Collected

MentorMod processes the following data when a rule violation is detected:
- Message content
- Message ID
- User ID
- Server ID

MentorMod does not collect data outside servers where it is installed.

## 2. Purpose of Processing

Data is processed for:
- Evaluating messages for rule compliance
- Enforcing server moderation rules
- Improving MentorMod functionality

## 3. Message Evaluation

Message content is sent to a third party AI service for violation evaluation.
MentorMod does not control how these providers process or retain data. Their handling of data is governed by their own terms and privacy policies.

## 4. Data Retention

Violation logs are currently retained indefinitely.
Future versions may introduce data retention limits or deletion mechanisms.

## 5. Feedback Data

If a user submits feedback using /feedback, the content is reviewed by the MentorMod team for product improvement purposes.
Feedback does not affect moderation actions.

## 6. Data Sharing

MentorMod does not sell or share collected data with third parties except for message evaluation services required to operate the bot.

## 7. Minors

MentorMod is intended for general audiences, including minors.
Server administrators are responsible for informing members about the presence and behavior of moderation bots.

## 8. User Rights

Users may contact the service owner with questions regarding data handling.
At this stage, individual data deletion requests may not be technically supported.

## 9. Changes to This Policy

This Privacy Policy may be updated over time. Continued use of MentorMod constitutes acceptance of the updated policy.
